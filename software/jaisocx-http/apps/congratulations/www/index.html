What your Jaisocx can do

Secure

Easiest way to provide a secured HTTPS connection to your site, is with our SSL setup script.

JS Frameworks

The server publishes easily and fast builds of any JS framework, also using cache and deflation of static js and css files for single-page applications.

Caching

Fast response due to stored in-memory cached resource found by urls index.

Gzip/deflate

Fast response due to up to 90% lower traffic thanks to deflation of textual resources hosted on server.

PHP

The PHP support was successfully tested with the ORO, the Symfony based complex application, Wordpress CMS and Magento internet-shop applications.

Video/Audio

Clever fast streaming of video and audio contents.

SQL-to-JSON

No need to write a controller and model, if you just wish to get an SQL statement result as a JSON response. Just provide an SQL statement with optional request parameters in the config file.

URLs Index

Fast in-memory b-tree index for resolving incoming urls to bound server's resources

Charset detection

Automatically transforming entire textual data into specified once charset in the config.
